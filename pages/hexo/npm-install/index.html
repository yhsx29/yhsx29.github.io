NPM 的安装与使用

介绍

NPM是Node.js的包管理工具。

安装

官方网址：https://nodejs.org/

下载地址：https://nodejs.org/en/download/

安装过程基本直接“NEXT”就可以了。

（windows的安装msi文件在过程中会直接添加path的系统变量，变量值是你的安装路径，例如“H:\nodejs”）。

如果没有变量，cmd 运行 npm 指令会失败

测试

安装完成后可以使用cmd（win+r然后输入cmd进入）测试下是否安装成功。

方法：在cmd下输入node -v，出现下图版本提示就是完成了NodeJS的安装。

配置

​ 模块路径、缓存路径

  1. 在安装目录 (D:\tools\nodejs)下 新建 node_cache 和 node_global 两个文件夹

  2. 打开 cmd,

    输入 npm config set prefix "D:\tools\nodejs\node_global" ，配置模块路径，

    输入 npm config set cache "D:\tools\nodejs\node_cache"， 配置缓存路径,

    输入 npm config set registry=http://registry.npm.taobao.org ，配置淘宝镜像，提高下载速度

    输入 npm config list 查看所有配置，

    完成后会在 C盘的用户目录下生成 .npmrc。

  3. 输入 npm install npm -g 重新安装 npm 移动到 node_global 文件下

  4. 配置环境变量 Path 添加 D:\tools\nodejs\node_global,新增NODE_PATH 值为 D:\tools\nodejs\node_global\node_modules

  5. 输入npm install vue -g 安装vue 到node_global 下
