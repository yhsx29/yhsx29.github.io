安装 hexo

hexo 官网

  1. 打开cmd,输入：npm install -g hexo-cli 。

  2. 创建一个工作目录 (我的是：D:\tools\doc\hexo)，打开cmd,进入到工作目录，输入：hexo init 进行初始化

目录结构:

  1. 配置_config.yml：

  1. 输入 hexo n demo 创建新的文件 demo.md,文件路径为 source/_post/demo.md,编辑md。（MarkDown的使用）

  2. 输入 hexo g 生成静态文件，文件存放在 public 文件夹下

  3. 输入 hexo s 本地服务启动，打开浏览器，输入地址：http://localhost:4000/ 即可浏览

  4. 输入 hexo d 部署到仓库中

下载Butterfly样式

  1. git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

    打开themes/butterfly 查看结构

  1. 创建自已配置：将默认配置剪切到 source/_data 路径下，修改名称为 butterfly.yml(防止更新主题时，配置被覆盖，更新后要及除默认配置，因为默认配置优先级高)

  2. 打开站点的_config.yml配置文件，将里面的 theme: landscape 改为 theme: butterfly 即可。

  3. 打开``butterfly.yml`配置 butterfly:

  1. 创建标签，分类 ，链接，关于 主页

    1. 打开cmd到工作空间下，输入
    1. 打开标签、分类、链接 主页，在 Front-matter 分别添加 type:'tags'、 type:'categories'、 type:'link' 属性

    1. 链接主页还需要创建 source\_data\link.yml

    配置

    1. 关于 主页需要自己手动编写

    2. 配置 404, butterfly.uml 添加配置激活内置404页面：本地不会跳转，要查看可以直接访问：http://localhost:4000/404.html

更多butterfly配置，请参考butterfly 文档
